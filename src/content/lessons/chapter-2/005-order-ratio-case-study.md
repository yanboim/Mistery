✅案例 1：委比 + 量比放大 → 买入信号

  - 某股股价在低位横盘，委比从 +10% 升至 +65%，量比从 1.2 升至3.0。
  - 表明有大量买盘涌入，股价可能突破平台，进入拉升阶段。

❌案例 2：委比 + 股价滞涨 → 诱多陷阱

  - 某股股价快速拉升，但委比维持在 +80%，股价未能突破前高。
  - 可能是主力诱多，需警惕冲高回落风险。

❌案例 3：负委比 + 低位止跌 → 筑底信号

  - 某股在连续下跌后，委比维持在 -30%，但股价未再创新低。
  - 可能是主力吸筹，形成潜在底部。

**委比的操作策略**

| 委比信号            | 操作策略             |
| --------------- | ---------------- |
| 委比 > +50%，且量比扩大 | 可能是突破信号，积极做多     |
| 委比 < -50%，且量比扩大 | 可能是恐慌性抛售，考虑止损或观望 |
| 委比 < 0，但股价不跌    | 可能是吸筹阶段，考虑低吸     |
| 委比 > 0，但股价不涨    | 可能是诱多陷阱，谨慎追涨     |
