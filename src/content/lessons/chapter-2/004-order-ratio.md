委比是股市中用来衡量买卖盘力量对比的一个重要指标，反映了市场在某一时刻买方力量与卖方力量的相对强弱，常用于判断市场的短期趋势和多空力量的博弈情况。

### 委比的市场含义

委比 > 0 → 买盘强于卖盘

  - 买单数量多于卖单，表明市场有更多的承接力量，可能推动股价上涨。

  - 委比越高，说明买方力量越强，可能有主力吸筹或资金抢筹的迹象。

委比 < 0 → 卖盘强于买盘

  - 卖单数量多于买单，表明市场抛压较大，可能引发股价下跌。
  - 委比越低，说明卖方力量越强，可能是主力出货或市场恐慌性抛售。

委比 = 0 → 买卖力量平衡

  - 市场处于平衡状态，可能是盘整或趋势即将选择方向的信号。

### 委比的市场信号

（1）高委比 + 股价上涨 → 看涨信号

  - 如果委比在 +50% 以上，且股价同步上涨，说明有大量买盘支撑，可能是主力吸筹或趋势启动。

  案例：
  - 委比从 +20% 升至 +60%
  - 股价同步突破重要压力位
  
    可能进入主升浪阶段

（2）低委比 + 股价下跌 → 看跌信号

  - 如果委比在 -50% 以下，且股价同步下跌，说明市场抛压沉重，可能是主力出货或市场恐慌。

  案例：
  - 委比从 -10% 降至 -70%
  - 股价跌破支撑位
  
    可能进入加速下跌阶段

（3）高委比 + 股价滞涨 → 可能是假突破

  - 如果委比在 +50% 以上，但股价未能有效突破压力位，可能是主力在制造买盘假象，诱导散户跟风。

  案例：
  - 委比维持在 +60%
  - 股价反复冲高但无法突破
  
    可能是诱多陷阱

（4）低委比 + 股价不跌 → 可能是筑底信号

  - 如果委比为负（如 -40%），但股价在低位稳定或有小幅反弹，可能是主力在低位吸筹。

  案例：
  - 委比维持在 -30%
  - 股价未创新低
  
    可能是吸筹阶段

### 委比与其他指标的配合

（1）委比 + 量比

  - 委比 > 0 + 量比扩大 → 买方力量增加，短期看涨
  - 委比 < 0 + 量比缩小 → 市场交投冷淡，趋势可能调整

（2）委比 + 换手率

  - 委比 > 0 + 换手率高 → 市场活跃，可能是新资金介入
  - 委比 < 0 + 换手率低 → 市场抛压不大，可能是整理行情

（3）委比 + K 线

  - 委比为正 + 长阳线 → 市场强势，主力可能在拉升
  - 委比为负 + 长阴线 → 市场恐慌，主力可能在出货
